这份SQLite3命令的中英文对照表，涵盖了**SQL命令**和**点命令（Dot Commands）** 两大类。SQL命令（如`SELECT`、`CREATE`）用于操作数据，以分号（;）结尾；点命令则用于管理数据库环境，以点（.）开头。

### 🛠️ SQLite3 点命令 (Dot Commands) 中英对照

| 命令 (Command) | 功能描述 (Description) |
| :--- | :--- |
| `.backup ?DB? FILE` | 备份 `DB` 数据库（默认为 "main"）到 `FILE` 文件。 |
| `.bail ON\|OFF` | 设置发生错误后是否停止执行。默认为 OFF。 |
| `.databases` | 列出当前所有已附加数据库的名称和文件。 |
| `.dump ?TABLE?` | 将数据库（或指定`TABLE`表）以SQL文本格式转储。 |
| `.echo ON\|OFF` | 开启或关闭命令回显（echo）。 |
| `.exit` | 退出 SQLite 交互式提示符。 |
| `.explain ON\|OFF` | 开启或关闭适合 `EXPLAIN` 的输出模式。 |
| `.headers ON\|OFF` | 开启或关闭查询结果的列名头部显示。 |
| `.help` | 显示所有点命令的帮助信息。 |
| `.import FILE TABLE` | 从 `FILE` 文件导入数据到 `TABLE` 表。 |
| `.indices ?TABLE?` | 显示所有索引的名称。若指定 `TABLE`，则只显示该表的索引。 |
| `.load FILE ?ENTRY?` | 加载一个外部扩展库。 |
| `.log FILE\|off` | 开启或关闭日志，可将日志输出到 `FILE`、`stdout` 或 `stderr`。 |
| `.mode MODE` | 设置查询结果的输出模式，如 `csv`, `column`, `html`, `insert`, `line`, `list`, `tabs`, `tcl` 等。 |
| `.nullvalue STRING` | 当查询结果为 NULL 时，用指定的 `STRING` 字符串代替显示。 |
| `.output FILENAME` | 将后续查询的输出重定向到 `FILENAME` 文件。 |
| `.output stdout` | 将输出重定向回屏幕（标准输出）。 |
| `.print STRING...` | 原样输出 `STRING` 字符串。 |
| `.prompt MAIN CONTINUE` | 替换标准的 `sqlite>` 提示符。 |
| `.quit` | 退出 SQLite 提示符（与 `.exit` 功能相同）。 |
| `.read FILENAME` | 执行 `FILENAME` 文件中的 SQL 语句。 |
| `.restore ?DB? FILE` | 从 `FILE` 文件恢复数据库到 `DB`（默认为 "main"）。 |
| `.schema ?TABLE?` | 显示表的 `CREATE` 语句（表结构）。若指定 `TABLE`，则只显示该表。 |
| `.separator STRING` | 改变输出模式和 `.import` 命令使用的分隔符。 |
| `.show` | 显示当前各种设置（如模式、分隔符等）的值。 |
| `.stats ON\|OFF` | 开启或关闭数据库统计信息的显示。 |
| `.tables ?PATTERN?` | 列出所有表名。若指定 `PATTERN`，则只列出匹配该模式的表。 |
| `.timeout MS` | 设置尝试打开锁定表的等待时间（毫秒）。 |
| `.timer ON\|OFF` | 开启或关闭 CPU 计时器，用于测量查询执行时间。 |
| `.width NUM NUM` | 为 `.mode column` 输出模式设置列宽。 |

### 🗄️ SQLite3 SQL 命令 (SQL Commands) 中英对照

| 命令 (Command) | 功能描述 (Description) |
| :--- | :--- |
| `ALTER` | 修改数据库中的已有对象（如表）的结构。 |
| `BEGIN` | 开始一个事务。 |
| `COMMIT` | 提交当前事务。 |
| `CREATE INDEX` | 在表上创建一个索引。 |
| `CREATE TABLE` | 在数据库中创建一个新表。 |
| `DELETE` | 从表中删除记录。 |
| `DROP TABLE` | 删除整个表及其数据。 |
| `INSERT` | 向表中插入新记录。 |
| `ROLLBACK` | 回滚当前事务。 |
| `SELECT` | 从一个或多个表中查询检索记录。 |
| `UPDATE` | 修改表中的现有记录。 |

### 💡 实用技巧

*   **查看帮助**：在 `sqlite>` 提示符下输入 `.help` 即可查看所有点命令。
*   **配置文件**：可将 `.headers on`、`.mode column` 等常用设置保存在 `~/.sqliterc` (Linux/macOS) 或 `%APPDATA%\sqlite\sqliterc` (Windows) 文件中，每次启动时会自动加载。
*   **命令行操作**：无需进入交互模式，可直接在终端执行命令，例如 `sqlite3 mydb.db "SELECT * FROM users;"`。
